# Networking-and-Content-Delivery
#### A Network is two or more client machines connected to each other.
#### Require a network device (i.e. a router) to connect.

#### Purpose
- To communicate and share resources.

#### LAN: Local Area Network (i.e. Home, Corporate Networks)
#### WAN: Wide Area Network ( i.e. The Internet)
#### Subnets: A network inside of a network, enables networks to perform more efficiently.

### Elastic Network Interfaces
- A primary private IPV4 address from the IPV4 address range of your VPE.

### (NAT) Network Address Translation
- NAT device enables instances in a private subnet to connect to the internet.
- NAT tables are basically DNS servers for private networks.

### Shared VPC's
- Separation of duties 
- Ownership 
- Security Groups 
- Effectiveness
- Optimized Costs

#### VPN: Virtual Private Network
- A connection from one private network to another private network, using a secure virtual tunnel.

#### AWS Direct Connect - Can be accessed without involved internet.

### VPC Endpoints (3 types) 
- Endpoint Service
- Gateway Endpoint
- Interface Endpoint

### AWS Transit Gateways
- A network transit hun that we can use to interconnect our VPC's and onsite networks.
